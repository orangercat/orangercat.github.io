Python基础 import与from…import…

############################

导入mode，import与from…import的不同之处在于，简单说：

如果你想要直接输入argv变量到你的程序中而每次使用它时又不想打sys，

则可使用：from sys import argv

一般说来，应该避免使用from..import而使用import语句，

因为这样可以使你的程序更加易读，也可以避免名称的冲突
