Blog架设在github pages

建立新的repository

建立username.github.io的repository

安装hexo

安装git,与node.js
$ npm install -g hexo

hexo相关文档

部署hexo

建立需要存放blog的文件夹,运行hexo初始化命令
$ hexo init

hexo相关命令

清除缓存文件 (db.json) 和已生成的静态文件 (public)。
$hexo clean $hexo c
生成静态文件
$hexo generate $hexo g
启动服务器。默认情况下，访问网址为：http://localhost:4000/。
$ hexo server
部署网站
$ hexo deploy

使用hexo主题NexT

下载主题

git更新NexT代码
$ git clone https://github.com/iissnan/hexo-theme-next themes/next

启用主题

修改hexo文件夹下_config.yml
theme: next

主题设定

在themes\next文件夹下_config.yml主题配置文件可以设定主题
scheme: Pisces
